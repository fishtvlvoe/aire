# VideoGo 安裝指南

## 安裝步驟（5 分鐘）

### Step 1：下載程式

打開終端機（Mac 叫「終端機」，Windows 叫「命令提示字元」），貼上這一行：

```bash
git clone https://TOKEN@github.com/fishtvlvoe/real-estate-AI.git
```

> 把 `TOKEN` 換成我們提供給您的授權碼

### Step 2：安裝

**Mac：**
```bash
cd real-estate-AI/videogo
chmod +x setup.sh
./setup.sh
```

**Windows：**
```bash
cd real-estate-AI\videogo
setup.bat
```

安裝程式會自動幫您裝好所有需要的工具。

### Step 3：設定 API Key

在 `videogo/` 資料夾裡建立一個叫 `.env` 的檔案，內容如下：

```
FAL_KEY=您的fal.ai金鑰
GOOGLE_AI_API_KEY=您的Gemini金鑰
```

**取得金鑰：**
- fal.ai 金鑰：到 https://fal.ai/dashboard/keys 註冊後取得
- Gemini 金鑰（免費）：到 https://aistudio.google.com/apikey 取得

### Step 4：確認安裝成功

```bash
npx tsx scripts/setup.ts
```

看到 `✔ 環境準備就緒！` 就表示安裝完成。

## 使用方式

### 生成影片

```bash
npx tsx scripts/pipeline-o3.ts
```

系統會詢問您：
1. 影片主題（例如：「為什麼房價一直下跌？」）
2. 影片長度（15 秒 / 30 秒 / 60 秒）
3. 場景描述（例如：「現代辦公室」）

然後自動生成影片，過程中會讓您確認每一步的產出。

### 批次生成

```bash
npx tsx scripts/queue-add.ts --topic "房價下跌的原因" --duration 15
npx tsx scripts/queue-add.ts --topic "升息對房貸的影響" --duration 15
npx tsx scripts/queue-loop.ts
```

## 常見問題

**Q：需要 OpenAI 的付費帳號嗎？**
A：不需要。腳本生成用 Google Gemini（免費），影片生成用 fal.ai。

**Q：可以換主播的照片嗎？**
A：可以。把新照片放到 `素材/主角/` 目錄，替換 `女主角.png`。

**Q：影片可以更長嗎？**
A：可以。目前支援 15 秒到 60 秒，更長的影片會自動分段生成再拼接。

**Q：Windows 和 Mac 都能用嗎？**
A：都可以。安裝腳本會自動處理。
