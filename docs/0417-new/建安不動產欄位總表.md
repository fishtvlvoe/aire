# 建安不動產欄位總表

## 使用原則
- 目的：供後續不動產說明書、物調表、591 PO 文、DM 文案、FB 導流貼文生成時對位。
- 標記：固定顯示 / 需估算 / 留空回填。
- 來源優先：謄本、合約、現場必問、秘書後補、公開行情資料。
