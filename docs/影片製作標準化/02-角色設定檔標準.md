# 角色設定檔標準（Character Profile SSOT）

> 版本：1.0
> 日期：2026-04-02
> 原則：所有影片生成參數從此檔案讀取，不在程式碼裡 hardcode

---

## 設定檔位置

```
config/characters/{角色名}/
├── character-profile.yaml    ← SSOT（單一真相來源）
├── character-final.png       ← 定妝照（FLUX Kontext 生成，已在場景裡）
├── voice-prompt.txt          ← 聲音描述（建立 voice_id 用）
└── references/               ← Element Binding 用的多角度照片
    ├── front.png
    ├── side-left.png
    └── side-right.png
```

## character-profile.yaml 格式

```yaml
# === 基本資訊 ===
identity:
  name: 沈知序
  age: 25
  persona: "高情緒價值房仲，自然成交型。她不是最強，但最讓人安心。"
  label: "calm confident Taiwanese woman"  # 用於 [Character: label] 標籤

# === Kling IDs（一次性建立，永久使用）===
kling_ids:
  element_id: ""          # Element Binding（臉部鎖定）
  voice_id: "868344520725897222"  # Create Voice（聲音鎖定）

# === SCALE Prompt 分層（固定不變）===
prompt_layers:
  # S — Shot（鏡位 + 運鏡）
  shot: >
    Medium shot from waist up. Static camera, no camera movement,
    no zoom, no dolly, fixed framing, tripod shot.

  # C — Character（角色 + 外貌 + 臉部控制）
  character: >
    A 25-year-old Taiwanese woman in navy blue business suit with
    white blouse stands in a warm modern living room facing camera.
    Composed and elegant, minimal facial movement, sincere and
    knowledgeable, speaks with quiet confidence, subtle warmth
    without exaggerated expressions, natural resting gentle smile,
    calm demeanor, refined temperament, not performative.
    Completely relaxed jaw — NO visible masseter tension, NO jaw
    clenching. Mouth opens only slightly when speaking, lips move
    gently with minimal effort. NO wide mouth opening, NO teeth-baring
    expressions. Natural effortless mouth movement as if casually
    talking in real life.

  # A — Action（動作，通常由腳本的 visualNotes 填入）
  action_default: >
    She speaks directly to the camera with calm steady eye contact.

  # L — Lighting & Location（燈光 + 場景 + 色調）
  lighting: >
    Cool blue cinematic color tone. Slightly desaturated, muted warm
    lighting with blue shadow tint. Elegant and calm visual mood.
    Soft natural lighting, no dramatic shadows.

  # E — Extra（技術規格）
  extra: >
    After finishing, she maintains a gentle composed expression.
    9:16 vertical format.

# === API 固定參數 ===
api_params:
  aspect_ratio: "9:16"
  generate_audio: true
  model: "fal-ai/kling-video/o3/standard/image-to-video"

# === 後製（理想狀態下不需要）===
post_processing:
  color_grade: null  # 色調由 prompt lighting 層控制
  resize: null       # 尺寸由 API aspect_ratio 控制
```

## Prompt 組裝規則（系統自動，人不介入）

```
最終 prompt = 拼接：
  {shot}
  {character}
  [Character: {label}, {expression_hint}]: "{narration}"
  {action 或 action_default}
  {lighting}
  {extra}
  <<<voice_1>>>
```

### 變數說明

| 變數 | 來源 | 範例 |
|------|------|------|
| `{narration}` | 腳本的台詞 | "看房找小序，生活有秩序" |
| `{expression_hint}` | 腳本的 expression 欄位 | "warm friendly smile" |
| `{action}` | 腳本的 visualNotes（如有） | "She gestures toward the window" |

### 不可變的部分

shot、character、lighting、extra、api_params — 這些從 character-profile.yaml 讀取，每次生成都一樣。改架構不能改這些。

### 可變的部分

narration、expression_hint、action — 每支影片不同，從腳本帶入。

## 驗證清單

生成影片後，對照此清單確認品質：

- [ ] 臉部與定妝照一致（Element Binding 有效）
- [ ] 嘴巴自然、下顎放鬆、無咀嚼肌過度用力
- [ ] 聲音與 voice_id 一致
- [ ] 色調為冷藍調（不需 FFmpeg 後製）
- [ ] 尺寸為 1080x1920（9:16 直出）
- [ ] 表情符合角色個性（平靜、自信、不誇張）
