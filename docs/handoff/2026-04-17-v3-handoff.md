## 📋 專案核心（新 session 必讀）

### 產品定位
- Docker 容器包，客戶 Windows 電腦裝 Docker Desktop 即可用
- 賣給房仲店家，取代電秘書文書工作（估 10-30 萬/台）
- 用客戶自備 Codex CLI 帳號（OpenAI 訂閱），不收 API key、資料不外流

### 三階段
1. **委託前**：業務輸入屋主姓名/地址/地號 → 系統自動查公開資料（v3 第一版 UI 做，爬蟲逐步 port）
2. **簽委託**：人工環節，系統不介入
3. **委託後**：選 13 種物件類型 → 填現場必問 + 秘書後補 → 一鍵產 5 份文件

### 5 份文件（輸出格式決策）
| 文件 | 格式 |
|------|------|
| 不動產說明書 | PDF（建安固定版面，建物/土地各一套） |
| 物調表 | MD |
| 591 PO 文 | MD |
| 銷售 DM 文案 | MD |
| FB 5 平台貼文 + 圖片提示詞 | MD |

### 13 種物件類型（全實作）
- 建物類（7）：公寓、大樓華廈、透天別墅、套房、店面、廠房、農舍
- 土地類（6）：農地、建地/住宅地、商業地、工業地、鄉村區建地、其他土地
