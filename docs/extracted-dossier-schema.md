# 不動產說明書（0417-old）章節結構萃取

## 1) 16 份 PDF 判斷

結論：**較像「同一份說明書/附件包被拆成多張單頁（模板/表單）」**，不是 16 個不同物件。

依據（從 OCR 可見的結構特徵）：
- 多份 PDF 內頁出現**連續的題號/頁碼**（例：現況調查表 p1~p4；稅費/規費 p5；增值稅概算、成交行情、周遭設施等固定段落）。
- 多數頁面是**空白欄位/條款/問卷**型態，缺少可區分物件的「完整門牌/地號/建號」填值（更像同一套文件的不同頁）。
- 其中 **不動產說明書3/16** 為「透明房價一覽表/成交行情」類附件（公開/市況資料），可視為附錄而非不同物件。

## 3) 建物版 vs 土地版：章節結構歸納

### 建物版（推定章節骨架）
- 封面/不動產說明書
- 法規告知與聲明（契約一部分、以謄本為準…）
- 產權調查表
  - 建物標示（建號、門牌、樓層、用途、主附建/公設面積…）
  - 土地標示（地號、地目、面積、分區…；若為建物亦會附土地）
  - 他項權利/限制登記（抵押權、查封、預告登記等）
- 稅費/規費/其他費用（含增值稅概算、地價稅等）
- 成交行情/周遭設施（附錄）

### 土地版（推定章節骨架）
- 封面/不動產說明書
- 法規告知與聲明
- 產權調查表
  - 土地標示（地號、地目、面積、使用分區/使用地類別、公告地價/現值…）
  - 權利/他項權利
- **基地/土地現況調查表（p1~p4…）**
  - 排水/河川/排水設施範圍、重要環境設施、國家公園/特別管制區
  - 建蔽率/法規相關核對
- 稅費/規費/其他費用（含增值稅概算）
- 成交行情/周遭設施（附錄）

## 4) 欄位來源：AI 產出 vs 謄本/表單取得

|欄位/段落|來源判斷|典型證據/說明|
|---|---|---|
|建物標示、土地標示、權利範圍、他項權利/限制登記|謄本/表單取得|屬「建物/土地登記謄本」原生欄位（地號/建號/面積/權利等）|
|使用分區/使用地類別、公告地價/公告現值|謄本/公開資料|地政/都計/地價公開資料或謄本欄位|
|稅費/規費清單（項目本身）|表單模板|固定欄位/選項（本批多為未填值表單）|
|增值稅「概算」數字|系統計算（非生成式 AI）|依公告地價/現值、持分、期間、物價指數等規則推算|
|現況調查表（問卷題目）|表單模板 + 人工/現場填答|題目固定；答案通常需現勘或委託人提供|
|成交行情/透明房價一覽表|外部資料（市況/實價）|非謄本；屬實價登錄/行情彙整附件|
|風險提示、重點摘要、價格建議、白話解釋|AI 產出（若系統有生成）|屬自然語言「解讀/建議」段落，非登記/表單原生欄位|
