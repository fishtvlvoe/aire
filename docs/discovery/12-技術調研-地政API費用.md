| 電子謄本第二類 | NT$20/張 | 完整資訊含抵押金額 |

## 好消息：有免費查詢方式

在全國地政電子謄本系統（ep.land.nat.gov.tw）中，點「**他項權利人查詢**」按鈕：
- **完全免費**
- 可看到：抵押權人姓氏/銀行名稱
- 看不到：具體抵押金額
- 不需要花錢申請謄本

**如果只是要確認「有沒有抵押」→ 免費就能做到。**
**如果要看「抵押多少錢」→ 最低 NT$20 一張謄本。**

## 各管道詳細

### 1. 全國地政電子謄本系統

| 項目 | 費用 |
|------|------|
| 電子謄本（土地/建物/地價/地籍圖） | NT$20/張 |
| 異動索引、異動清冊 | NT$20/張 |
| 建物門牌查詢地建號 | NT$10/筆 |
| 超商 ibon 額外手續費 | NT$10 + 列印費 |

- 他項權利部包含在謄本中，不另計費
- 第二類謄本任何人可申請（會隱匿部分個資，但抵押權資訊看得到）
- 免費客服：0800-080-212

### 2. 地政資訊 e 點通

| 項目 | 費用 |
|------|------|
| 建物門牌查詢地建號 | NT$10/筆 |
| 電子謄本 | NT$20/張 |
| 人工登記簿謄本 | NT$10/張 |

- 不是點數制，是直接收費
- 中華電信客戶：費用併入帳單
- 關貿網路客戶：按月寄帳單

### 3. 地政整合資訊服務共享協作平台 API

- 單筆計價、使用者付費
- 分為免費介接（8 項）和付費介接（10 項）
- **具體單價未公開，必須打電話問**
- 已有永慶房屋、國泰建設等業者使用中
- 月均支付從「3 位數成長至 5 位數」（數百元 → 數萬元，視用量而定）

**聯絡方式：**
- 電話：(04) 2451-6609 分機 515 或 503
- Email：jillhuang@gis.tw 或 envie@gis.tw
- 服務時間：週一至五 09:00-11:45、13:00-17:30

## 對專案的影響

### 費用試算（以客戶月處理 100 個物件估算）

| 方案 | 做法 | 月費 |
|------|------|------|
| 最省方案 | 免費查有沒有抵押 + 客戶自己拍謄本給 AI 看 | NT$0 |
| 基本方案 | 免費查抵押 + 有疑慮的才花 $20 查謄本 | NT$200-400（約 10-20 筆） |
| 完整方案 | 每個物件都查電子謄本 | NT$2,000（100 筆 × $20） |
| API 自動化 | 串接共享協作平台 | 待確認（估 NT$3,000-10,000） |

### 建議

**MVP 階段用「最省方案」：**
1. 先用免費的「他項權利人查詢」確認有沒有抵押
2. 有疑慮的物件，客戶自己花 $20 拉謄本，拍照丟給系統
3. 等量起來了再考慮串接 API 自動化

**這樣地政查詢的費用幾乎是零。**
