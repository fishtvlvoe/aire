# 技術調研：實價登錄 API 與不動產資料串接

> 日期：2026-03-19
> 狀態：調研完成

## 結論摘要

| 資料來源 | 有沒有 API | 可行性 | 費用 |
|----------|-----------|--------|------|
| 實價登錄（內政部） | ❌ 無 REST API，但有 CSV 批次下載 | ✅ 可行 | 免費 |
| 政府開放資料平台 | ✅ 有資料集可程式化下載 | ✅ 可行 | 免費 |
| 地籍資料（謄本查詢） | ✅ 有付費 API | ✅ 可行 | 付費制 |
| 591 | ❌ 無官方 API | ⚠️ 爬蟲有法律風險 | N/A |
| 永慶/信義 | ❌ 無公開 API | ❌ 不可行 | N/A |

## 1. 實價登錄 — 批次下載方案（免費）

### 資料來源
- **下載系統**：https://plvr.land.moi.gov.tw/DownloadOpenData
- **格式**：CSV
- **更新頻率**：每月 1、11、21 日更新（約每 10 天）
- **授權**：政府資料開放授權條款第1版（免費）

### 資料欄位
- 鄉鎮市區、交易標的、土地位置/建物門牌
- 土地移轉總面積、使用分區
- 交易年月、移轉層次、總樓層數
- 建物型態、主要用途、建材、建築完成年月
- 建物移轉總面積、總價元、單價元/平方公尺

### 政府開放資料平台上的資料集

| 資料集 | 編號 | 說明 |
|--------|------|------|
| 本期買賣實價登錄批次資料 | 25119 | 當期資料，每 10 天更新 |
| 本期不動產實價登錄批次資料 | 77051 | 含買賣、租賃、預售屋 |
| 非本期實價批次資料 | 18703 | 歷史季度資料（2012 年起） |

### 各縣市開放資料平台
- 新北市：提供 JSON API（https://data.ntpc.gov.tw/）
- 高雄市：CSV 格式（https://data.kcg.gov.tw/）
- 臺中市：JSON/XML/CSV（https://opendata.taichung.gov.tw/）

### 建議技術路線

```
plvr.land.moi.gov.tw/DownloadOpenData
        ↓ (排程下載 CSV，每月 1/11/21 日)
    ETL 解析入庫
        ↓
   自建 PostgreSQL 資料庫
        ↓
   自建 REST API 對外服務
        ↓
   AI 分析引擎查詢
```

## 2. 地籍資料（謄本查詢）— 付費 API

### 地政整合資訊服務共享協作平台
- **新 URL**：https://cop.moi.gov.tw
- **API 基底 URL**：https://copapi.moi.gov.tw
- **Open API**：https://openapi.moi.gov.tw
- **使用資格**：民間企業和個人均可申請（2019 年 7 月起開放）
- **費用**：付費制，需簽授權合約

### 可用的 API 服務

| API 代碼 | 名稱 | 用途 |
|----------|------|------|
| MOI_API_003 | 地籍土地他項權利部資料 | ⭐ 查抵押權、假扣押 |
| MOI_API_007 | 地號資料查詢 | 查土地基本資訊 |
| MOI_API_012 | 全國土地基本資料庫代碼 | 23 個查詢介面 |
| MOI_API_024 | 地籍圖詮釋資料 | 地籍圖資訊 |

### 其他查詢管道
- 全國地政電子謄本系統：https://ep.land.nat.gov.tw/
- 地政資訊 e 點通：https://tttland2.tradevan.com.tw/（購買點數查詢）
- 地籍圖資網路便民服務系統：https://easymap.land.moi.gov.tw/（免費瀏覽，無 API）
- 客服電話：(04) 2451-6609

## 3. 591 — 不建議碰

- 無官方 API
- 有積極的反爬蟲機制（IP 封鎖、資料混淆）
- 爬取用於商業用途有法律風險
- **結論：不碰 591，用實價登錄替代**

## 對專案的影響

### ✅ 好消息
- 實價登錄免費且合法，可以直接用
- 地政 API 可以查他項權利（抵押、假扣押），客戶要的風險檢查技術上可行
- 有現成的開源專案可以參考

### ⚠️ 要注意
- 實價登錄資料延遲 1-3 個月（盲點分析提到的問題確認了）
- 地政 API 要付費 + 申請，時間和費用待確認
- 591 資料無法合法取得，「同區域目前開價」這個功能可能要換方式做
