AIRE 安裝指引

適用版本 v0.1.3 · 給第一次安裝的用戶

⊕ 確認你的電腦版本

Mac 用戶

點螢幕左上蘋果圖示 → 關於這台 Mac → 看「晶片」欄位：

  • 顯示 Apple M1 / M2 / M3 / M4 → 用 arm64 版本
  • 顯示 Intel → 用 x64 版本

Windows 用戶

直接使用 AIRE-setup-0.1.3.exe 即可。

（目前支援 Windows 10 / 11，64-bit）

⊕ Mac 安裝步驟
1
雙擊 DMG 檔案
出現一個視窗，把左邊的 AIRE 圖示拖到右邊的「Applications」資料夾。
2
第一次開啟
到「應用程式」資料夾找到 AIRE 雙擊。
⚠ Mac 第一次會跳警告：「無法驗證開發者」或「無法打開」

這是正常的（軟體還沒上架到 Mac App Store）。處理方式：

系統設定 → 隱私權與安全性 → 滑到中下方會看到「已封鎖 AIRE」→ 點 「仍要開啟」 → 輸入電腦密碼確認 → 重新打開即可。

⊕ Windows 安裝步驟
1
雙擊 AIRE-setup-0.1.3.exe
跟隨安裝精靈一路按「下一步」即可。
⚠ Windows 第一次會跳「Windows 已保護你的電腦」藍色畫面

處理方式：點 「其他資訊」 → 「仍要執行」。

⊕ 啟用流程（首次開啟必做）
1
安裝 Codex CLI
AIRE 需要 OpenAI 的 Codex 工具來驅動 AI 功能。首次開啟時，系統會自動偵測是否已安裝。
若尚未安裝，畫面會顯示引導頁面，請依照指示完成安裝：
  • Mac：打開「終端機」，貼上指令 npm install -g @openai/codex
  • Windows：打開「命令提示字元」或「PowerShell」，貼上同樣指令

（這一步通常由服務人員協助完成）

2
登入 ChatGPT 帳號
安裝 Codex CLI 後，在終端機輸入 codex login，瀏覽器會跳出 ChatGPT 登入頁面。
登入完成後回到 AIRE，點擊「重新偵測」即可。

（需有 ChatGPT 帳號，免費帳號即可使用；付費帳號效果更好）

3
輸入產品序號
畫面會出現「啟用」頁，輸入服務人員提供的序號（格式：XXXX-XXXX-XXXX-XXXX），按「啟用」。
系統會自動把這台電腦記下來（俗稱「綁機」），之後同一台電腦不用再啟用。每張序號只能綁一台電腦。
4
建立管理員帳號
設定一組電子信箱與密碼，這是登入 AIRE 的帳號（自己記好，後台找回密碼較麻煩）。
5
登入完成
回到登入頁，輸入剛建立的 Email 與密碼，進入 AIRE 主頁。可以開始上傳謄本、產出文件。
⊕ 常見狀況處理
狀況解決方式
序號顯示「已被使用」每張序號只能綁一台電腦。如要換電腦，請聯絡服務人員「解除綁定」。
忘記管理員密碼聯絡服務人員協助重設。
更新提示出現同意更新即可（系統會自動下載新版並重開）。
App 無法啟動確認 Mac 系統設定已點過「仍要開啟」、Windows 已點過「仍要執行」。
序號輸入後一直轉圈檢查網路連線，啟用流程需要連到 license 伺服器。
Codex CLI 安裝失敗確認電腦已安裝 Node.js（至少 v18）。可在終端機輸入 node -v 確認。若未安裝，請先到 nodejs.org 下載安裝。
codex login 沒有跳出瀏覽器手動打開瀏覽器到 chatgpt.com 登入後，再重新執行 codex login。
