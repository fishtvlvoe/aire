# 不動產物件分析助手 — 專案總覽

## 這是什麼？

一個 AI 工具，幫你在**簽委託之前**快速分析物件。

打一個地址，3 分鐘內拿到完整的分析報告。

## 你的痛點

- 手頭物件太多，沒時間一個一個分析
- 不確定屋主開價是否合理
- 需要數據來說服屋主調價
- 每次都要手動查實價登錄、上 591 比對

## 系統幫你做什麼

```
你打一個地址
    ↓ 系統自動幫你做（3 分鐘內）
    ├── 查實價登錄：附近 3 公里內近 1 年成交了幾筆、均價多少
    ├── 查周邊設施：最近的學校、醫院、公園、車站
    ├── 估合理價格：根據成交資料算出建議售價區間
    ├── 寫議價策略：如果屋主開太高，怎麼跟他談
    ├── 做優劣分析：這個物件的賣點和缺點
    ├── 給經營建議：簽委託後怎麼跟屋主溝通
    └── 推連結給你：591、好房網、永慶、信義的同區物件搜尋
    ↓
    你拿到一份報告
    ↓
    帶著報告去跟屋主談
```

## 你需要做什麼？

**打一個地址就好。** 其他全部交給系統。

如果你想要更完整的分析，可以額外做（但不是必要的）：

- 點系統給你的 591 連結 → 看一下同區開價 → 截圖傳回來
- 查一下謄本有沒有抵押 → 告訴系統有或沒有
- 輸入你自己的判斷（例如「屋況不錯」「屋主急售」）

## 報告長什麼樣？

像 ChatGPT 的對話，一段一段出來，每個數字都可以點開看資料來源。

不是 PDF，不用列印。手機打開就能看。

## 這個系統不是什麼

| 不是 | 是 |
| --- | --- |
| 不是自動賣房系統 | 是分析助手，幫你看得更清楚 |
| 不是估價報告 | 是市場參考分析 |
| 不是取代你的判斷 | 是放大你的判斷力 |
| 不是給屋主看的 | 是你自己的操作手冊 |
