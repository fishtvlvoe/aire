# 市場分析摘要 — 為什麼值得做？

## 台灣市場現況

### 你要做的事，目前沒有人在做

| 功能 | 市場上有嗎？ |
|------|-------------|
| AI 開價合理性分析 | ❌ 沒有 |
| AI 議價策略生成 | ❌ 沒有 |
| 物件風險自動檢查 | ❌ 沒有 |
| AI 寫行銷文案 | ✅ 住商不動產有（但只有文案，沒有分析） |
| AI 看房（3D/VR） | ✅ 信義房屋有（但那是給買方用的） |

**重點：大型房仲的 AI 都是做給「買房的人」用的。沒有人做給「房仲自己」用的決策工具。**

## 技術可行性

### 資料來源確認

| 資料 | 有沒有？ | 費用 |
|------|---------|------|
| 實價登錄歷史成交 | ✅ 有，政府公開資料 | 免費 |
| 周邊設施（學校、醫院等） | ✅ 有，Google Maps API | 免費（用量內） |
| 行政區/地段資訊 | ✅ 有，國土測繪中心 API | 免費 |
| 591 同區開價 | ⚠️ 無法自動取得 | 系統生成連結，你自己看 |
| 抵押/假扣押 | ⚠️ 無法自動取得 | 你自己查謄本，結果告訴系統 |

### 核心結論

- 最重要的資料（實價登錄）免費且合法
- AI 分析的數字不是亂猜的，都來自真實成交紀錄
- 每個數字都可以追溯來源

## 風險與限制

| 限制 | 怎麼處理 |
|------|---------|
| 實價登錄資料延遲 1-3 個月 | 系統會標示資料時間，提醒你注意 |
| 冷門區域成交少 | 資料不足時會警告，不硬給數字 |
| 系統不能查凶宅/違建 | 明確告訴你「這些沒查」，你要自己判斷 |
| 不是正式估價報告 | 報告上會寫「市場參考分析，僅供參考」 |
