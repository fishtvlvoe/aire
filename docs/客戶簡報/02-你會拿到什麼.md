# 你每分析一個物件，會拿到什麼？

## 自動產出的（打一個地址就有）

### 1. 物件基本資訊
- 座標位置
- 行政區 / 里
- 地段

### 2. 實價登錄分析
- 附近 3 公里內，近 1 年有幾筆成交
- 成交均價（例：850 萬）
- 成交中位數（例：830 萬）
- 每坪單價範圍（例：18.5 萬 ~ 22.3 萬/坪）
- 價格趨勢（例：近 3 季下跌 5.2%）
- 每一筆原始成交紀錄（地址、坪數、總價、日期）
- 資料是什麼時候的（系統會告訴你）
- 資料夠不夠（不夠會警告你）

### 3. 周邊設施
- 最近的學校（名稱 + 幾公尺）
- 最近的醫院
- 最近的公園
- 最近的車站
- 最近的超市/賣場
- 最近的警局
- 生活機能評分（1-10 分）

### 4. 市場價格預估
- 建議售價區間（例：830 萬 ~ 900 萬）
- 如果你輸入了屋主開價，會告訴你高於行情多少 %
- 以這個開價預估要賣多久（例：3-6 個月）
- 每個數字都可以點開看是怎麼算的

### 5. 議價策略
- 開價合不合理的判斷
- 建議的出價區間
- 三種不同的談判話術
- 買方心理分析（這個價位的買家在意什麼）
- 如果屋主堅持不降價，會發生什麼

### 6. 優劣分析（SWOT）
- 這個物件的賣點（例：屋齡新、生活機能好）
- 這個物件的缺點（例：開價偏高、同區競爭多）
- 機會（例：附近有新建設）
- 威脅（例：房價下跌趨勢）
- 整體評估：建不建議接這個案子

### 7. 屋主經營指南
- 怎麼跟這種屋主溝通
- 建議多久回報一次
- 什麼時候建議降價
- 哪些事情要避免

### 8. 平台搜尋連結
- 591 售屋 → 點一下看同區在賣的房子
- 好房網 → 同上
- 永慶房屋 → 同上
- 信義房屋 → 同上
- Google 地圖 → 看周邊環境

## 你可以額外補充的（非必要，但會讓報告更完整）

### 9. 591 同區開價分析
- 你點連結去 591 看完後，截圖傳回來
- 系統會告訴你：目前開價 vs 實際成交價差多少

### 10. 物件風險檢查
- 你自己查謄本後，告訴系統有沒有抵押
- 如果有，系統會標示風險等級

### 11. 你的經驗判斷
- 你覺得值多少？系統會跟它的估價比較
- 你知道的額外資訊（屋主急售、附近要蓋捷運等）
