▋ 建安不動產物件管理系統 — 安裝說明

步驟1：安裝 Docker Desktop

請先安裝 Docker Desktop，官方下載連結是 https://www.docker.com/products/docker-desktop/。如果您使用的是 Windows，需為 Windows 10 或 Windows 11 的 64-bit 版本才支援安裝。

步驟2：下載系統檔案

請向顧問索取系統檔案，至少需要 `docker-compose.yml` 與 `.env` 兩個檔案，拿到後放在同一個資料夾中。

步驟3：建立 .env 檔

如果您拿到的是 `.env.example`，請先複製一份並改名為 `.env`，再把 `CODEX_API_KEY` 填入正確的金鑰內容。

步驟4：啟動系統

打開 Terminal，切換到系統檔案所在資料夾後，執行 `docker compose up`。第一次啟動大約需要等 1 分鐘。

步驟5：開啟系統

打開瀏覽器，前往 http://localhost:3000，即可進入系統畫面。

▋ 常見問題

• Docker 啟動失敗怎麼辦

先確認 Docker Desktop 是否已經開啟，並且狀態是正在執行中，再重新執行 `docker compose up`。

• 忘記網址怎麼辦

請直接輸入 http://localhost:3000。
